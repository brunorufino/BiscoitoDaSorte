Instruções

Clone o repositório:
git clone https://github.com/layout-padrao-react-native.git

Navegue até o diretório do projeto:
cd layout-padrao-react-native

Executar no termninal os comandos abaixo:

npm install

npx expo start 


Tecnologias utilizadas:
React Native
Expo
